Automatically start the video when switching to the tab:     

To keep the extension fast, there is no way to change this behaviour through options.

If you want the version without the automatic start, use Stop Autoplay for Youtube™ from the Chrome Webstore.

Any other problems? Report them on GitHub (This project is Open Source) or mail me kurtextrem[at]gmail.com.

If this extension is helpful for you, I would very much appreciate a rating in the Webstore.
I started developing this extension when I was 14 and I try everything to keep it updated, 5 years later.
